Elevator project for group w11 in TTK4145. Authors: Lars Gustavsen og Christian Fredrik Saetre.

To run the program change directory to main and run the command "go build main.go && ./main" and the program should start up.
Note that processpairs is active and wil launch a second terminal, so if you want to turn of the program remember to turn off the 
save terminal first then the program inn the main terminal (processpairs can only launch 3 times).
